The Box Model

Used to determine size, position, and properties of each element on a page.

Margin

Border

Padding

Content


Area Definition/Function CSS Properties
Content Contains content that should be viewd on page (ie. text, images, video player, etc.) width, height, box-sizing
Padding Extends blank area around content padding (shorthand: padding: top, right, bottom, left)
Border Extends padding area to include a border for the element border, border-width, height
Margin Blank area that extends border to seperate an element from other elements. margin (shorthand: margin: top, right, bottom, left)
